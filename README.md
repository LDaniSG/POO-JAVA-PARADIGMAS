# POO-JAVA-PARADIGMAS

Repositorio académico para explicar los paradigmas más importantes de la
Programación Orientada a Objetos en Java.

## Contenido
- 01-Encapsulación
- 02-Abstraccioón
- 03-Herencia
- 04-Polimorfismo
- 05-Asociación
- 06-Agregación
- 07-Composición

## Autor
Laura Daniela Cepeda Gamba

## Objetivo
Comprender y ejemplificar los principales paradigmas y relaciones de la POO
mediante aplicaciones sencillas hechas en Java.
